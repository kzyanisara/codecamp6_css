Property และ Value คืออะไรแตกต่างกันอย่างไร

Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ชนิดตัวอักษร , ขนาดตัวอักษร เช่น color: pink; color คือ property ถ้ามี property มากกว่า 1 เราจะใช้ ; คั่นระหว่าง property p { text - align : center ; color : red }
Value คือ คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , ตัวอักษรแบบ tahoma , ตัวอักษรขนาด 12 pt เช่น color: pink; pink คือ value

property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง

5 กลุ่ม

  • content: : color , size , font , text, list-style-type, Height, Width, background
  • border: border, table
  • padding: padding (top right buttom left)
  • margin: margin (top right buttom left)
  • element: Display, Height, Width, Position, Float, Align, Flex, Grid

    value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร

    มี 2 ประเภท

    1. Absolute lenghts คือ ค่าที่แน่นอน

  • cm เซนติเมตร
  • mm มิลลิเมตร
  • in นิ้ว = 96px = 2.54cm
  • px พิกเซล 1/96 นิ้ว
  • pt พอยท์ 1/72 นิ้ว
  • pc พิคาล 1/12 นิ้ว = 12pt
    		•

    2. Relative lenghts คือ ค่าที่เกี่ยวกับความสัมพันธ์

  • em เท่าขนาดฟอนต์ของ element
  • ex เท่าความสูงของฟอนต์ของ element
  • ch ความกว้างของตัวหนังสือใน parent element
  • rem เท่าขนาดของ root element
  • vh เปอร์เซนต์ความสูงของหน้าจอ
  • vw เปอร์เซนต์ความกว้างของหน้าจอ
  • vmin เปอร์เซนต์ของ view point ต่ำสุด
  • vmax เปอร์เซนต์ของ view point สูงสุด
  • % เปอร์เซนต์ของ parent element
