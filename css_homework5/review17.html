position คืออะไร และ อะไรบ้าง z-index คืออะไรใช้อย่างไร

position z-index

position คือ การจัดวางตำแหน่งของ css ประกอบไปด้วย

  1. static
  2. relative
  3. absolute
  4. fixed

z-index คือ ใช้จัดเรียง elementที่ซ้อนกัน และมีผลกับ element ที่กำหนด position ไว้เท่านั้น
(เช่น position:absolute; ,position:relative;)
