value ใน display property มีอะไรบ้าง

  • display: inline; //แสดงผลในบรรทัดเดียวกัน
  • display: block; //แสดงผลคนละบรรทัด
  • display: none; //ซ่อนการแสดงผล
  • visibility:hidden; //ซ่อนการแสดงผล แบบเหลือพื้นที่ไว้

    grid และ flex แตกต่างกันอย่างไร

    grid: การแบ่งพื้นที่ออกเป็นส่วนๆ พื้นที่หน้าจอจะถูกแบ่งได้ก็ต่อเมื่อมีเส้นตัดผ่านจากบนลงล่างและจากซ้ายไปขวาเกิดเป็นลักษณะตารางที่เรียกว่า Grid
    item ด้านในมีการจัดเรียงตัวที่ไม่ได้มีแค่แถวเดียว แต่มีลักษณะเป็น column กับ row
    flex: การออกแบบพื้นที่โดยการจัดเรียงไปแนวนอน หรือแนวตั้ง สามารถย้าย item ต่าง ๆ ได้ ช่วยให้การจัดกล่องต่างๆ ในหน้าเว็บ
    flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน

    justify and align แตกต่างกันอย่างไร

    align: จัดให้ชิดซ้าย ขวา หรือวางตรงกลางข้อความ ใช้ค่าเฉพาะคือ left , right, center , justify มีค่า Default คือ left จะไม่สามารถขยายขอบได้
    justify: คือจัดให้ทั้งซ้ายและขวา ตรงข้ามกันหมด โดยเพิ่มช่องว่างระหว่างคำเพื่อให้ทั้งสองด้านตรงกันหมด จะทำการจัดข้อความโดยสามารถขยายกรอบของข้อความได้
